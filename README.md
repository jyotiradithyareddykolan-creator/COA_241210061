# Computer Organization and Architecture (CSBB251) Lab Repository

This repository contains a collection of **Computer Organization and Architecture (COA)** laboratory experiments implemented and documented as part of academic coursework.

The experiments are designed to provide a strong foundation in:

- Digital Logic Design
- Computer Architecture Concepts
- Sequential and Combinational Circuits
- Low-Level Programming
- Debugging Techniques

---

# Repository Structure

The experiments are organized systematically for clarity and easy navigation.

---

## Experiment 01 – Basic Logic Circuits

Implementation of fundamental logic gates and combinational circuits.

### Topics Covered
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- Basic Combinational Circuits

---

## Experiment 02 – 4-bit Ripple Carry Adder

Design and simulation of a 4-bit Ripple Carry Adder using basic logic components.

### Topics Covered
- Half Adder
- Full Adder
- Ripple Carry Addition
- Binary Arithmetic

---

## Experiment 03 – GDB Debugger (Compiler Debugging)

Debugging C programs using the GNU Debugger (GDB).

### Topics Covered
- Breakpoints
- Step Execution
- Variable Inspection
- Stack, Queue, and Linked List Debugging

---

## Experiment 04 – 8-bit Shift Register using Flip-Flops

Design and implementation of an 8-bit Shift Register using D Flip-Flops.

### Topics Covered
- Shift Registers
- Sequential Circuits
- Clock Synchronization
- Bit Manipulation

---

## Experiment 05 – Addressing Modes

Implementation and execution of different addressing modes using the GDB compiler.

### Topics Covered
- Immediate Addressing
- Direct Addressing
- Indirect Addressing
- Indexed Addressing
- Relative Addressing
- Stack Addressing

---

## Experiment 06 – Shift Micro-Operations

Implementation of different shift micro-operations using Logisim simulator.

### Topics Covered
- Logical Shift
- Arithmetic Shift
- Rotate Shift
- Rotate with Carry
